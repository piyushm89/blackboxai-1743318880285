About This Project

College Event Management System

This system was designed to help colleges and universities manage their campus events more efficiently. It provides a complete solution for event creation, promotion, registration, and administration.

Features

  • Create and manage events with detailed information
  • Student registration system for events
  • Admin dashboard for event management
  • Responsive design that works on all devices
  • Local storage for data persistence
